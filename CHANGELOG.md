# Changelog

## 1.0.10

- Fixed iPhone keyboard text input so `Input.SendText` does not submit/go/enter after inserting text.
- Fixed iPhone bridge idle timeout handling in the Kodi add-on so harmless socket timeouts do not disconnect the bridge.
- Published `script.xbox.proxy-1.0.10.zip` to the static Kodi repository.

## 1.0.9

- Added iPhone Remote Bridge support for the custom `KodiXboxRemote` app.
- Added iPhone app source, Xcode project, and GitHub Actions unsigned IPA build workflow.
- Added settings persistence, haptics, text input, and local network listener support.

## Earlier releases

Earlier versions focused on Xbox Kodi install diagnostics, repository hosting, WebSocket stability, compression framing, and dashboard/proxy management.
