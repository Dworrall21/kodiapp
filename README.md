# Dworrall21 Kodi Repository

This is a static Kodi add-on repository hosted on GitHub Pages. It contains the **Xbox Web Proxy** service and diagnostic tools.

## How to Install the Repository in Kodi

To easily install and receive automatic updates for the add-ons in this repository, you need to add this source to Kodi and then install the repository add-on.

### Step 1: Add the File Source
1. Open Kodi.
2. Go to **Settings** (the gear icon at the top left).
3. Select **File manager**.
4. Double-click on **Add source**.
5. Click on `<None>` and enter the following exact URL:
   `https://dworrall21.github.io/kodiapp/`
6. Click **OK**.
7. Enter a name for this media source, such as `Dworrall21 Repo`.
8. Click **OK** to save the source.

### Step 2: Install the Repository Add-on
1. Go back to the Kodi home screen.
2. Select **Add-ons** from the left menu.
3. Click the **Add-on browser icon** (the open box icon at the top left).
4. Select **Install from zip file**.
   * *Note: If this is your first time installing from a third-party source, Kodi will prompt you to enable "Unknown sources" in Settings. Enable it and go back.*
5. Select the source you added in Step 1 (e.g., `Dworrall21 Repo`).
6. Select `repository.dworrall21-1.0.0.zip` to install.
7. Wait for the "Add-on installed" notification to appear in the top right corner.

### Step 3: Install Add-ons from the Repository
Now that the repository is installed, you can browse and install add-ons from it:
1. From the Add-on browser screen, select **Install from repository**.
2. Select **Dworrall21 Kodi Repository**.
3. Browse the categories (e.g., Services or Program add-ons) to find and install the add-ons you need, such as the **Xbox Web Proxy**.
